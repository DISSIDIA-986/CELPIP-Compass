## 🎨 设计规范 (最简化)

### 配色方案
```css
主色: #2563eb (蓝色 - 专业)
成功: #10b981 (绿色 - 学习完成)
警告: #f59e0b (橙色 - 重要提醒)
背景: #f9fafb (浅灰 - 护眼)
```

### 字体
```css
标题: 18px - 24px (加粗)
正文: 16px (正常)
说明: 14px (浅色)
```

### 组件样式
```css
卡片: 白色背景 + 微妙阴影 + 圆角8px
按钮: 蓝色背景 + 白色文字 + 圆角4px
标签: 灰色背景 + 圆角12px + 内边距4px
```

## 🔧 技术实现要点

### 1. 页面路由
```typescript
// 只需要4个页面
/          - 首页 (学习总览)
/techniques - 应试技巧 (核心)
/prompts    - AI Prompt工具 (核心)
/plan       - 学习计划 (辅助)
```

### 2. 状态管理 (最简单)
```typescript
// 只需要最基本的状态
interface UserState {
  name: string;
  targetLevel: number;
  currentLevel: number;
  completedTasks: string[];
  favoritePrompts: string[];
}
```

### 3. 数据结构 (极简)
```typescript
// 技巧数据
interface Technique {
  id: string;
  title: string;
  category: 'writing' | 'speaking' | 'listening' | 'reading';
  task?: number;
  duration: number; // 分钟
  improvement: number; // 预期提升分数
  points: string[];
}

// Prompt数据
interface Prompt {
  id: string;
  title: string;
  category: string;
  description: string;
  content: string;
  rating: number;
  usageCount: number;
  tags: string[];
}
```
