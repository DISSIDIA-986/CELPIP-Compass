# CELPIP 4周高强度备考计划
## CLB 7→8-9加速方案

**适用场景:** 急需参加考试的考生  
**时间投入:** 每周8-10小时 (vs 标准8周的4-5小时)  
**目标:** CLB 8-9 (争取综合8.5+)  
**难度:** ⭐⭐⭐⭐⭐ (高强度)

---

## 📊 对比: 4周 vs 8周

| 指标 | 4周方案 | 8周方案 | 差异 |
|------|--------|--------|------|
| 周投入 | 8-10小时 | 4-5小时 | 2倍密度 |
| 总投入 | 32-40小时 | 40-48小时 | 少8小时但密度高 |
| 每天 | 1.2-1.5小时 | 35-40分钟 | 强度更大 |
| 学习效率 | 需保持100%专注 | 可恢复期 | 更考验意志 |
| 预期结果 | CLB 8-9可达成 | CLB 8.5-9稳妥 | 4周风险更高 |

**关键:** 4周方案需要极高的专注度和执行力，不适合边工作边学习

---

## 🎯 4周战略概览

```
Week 1: 冲刺准备 (集中诊断 + 快速上手)
├─ 目标: 熟悉格式，确定弱点，建立节奏
├─ 学习时间: 8小时
├─ 关键: 速度优先
└─ 结果: 明确的弱点科目

Week 2: 写作突破 (高ROI科目深度)
├─ 目标: Task 1-2模板掌握，完成度100%
├─ 学习时间: 10小时
├─ 关键: 大量写作练习 (每天2-3篇)
└─ 结果: 写作CLB 7.5-8

Week 3: 融合冲刺 (全科平衡)
├─ 目标: 口语框架应用，听力Part 4掌握，阅读时间管理
├─ 学习时间: 10小时
├─ 关键: 科目平衡，开始完整练习
└─ 结果: 口语8，听力阅读7-7.5

Week 4: 最后冲刺 (模拟+调整)
├─ 目标: 2次完整模拟，薄弱点修复
├─ 学习时间: 8小时
├─ 关键: 心理建设，应试技巧
└─ 结果: 自评CLB 8.5+，准备好参考试
```


## ⚠️ 4周方案的风险

### 1. 执行风险 🔴 高

```
✗ 需要极高的自律性 (28天连续学习)
✗ 无恢复期，一旦出现倦怠会雪崩
✗ 无法应对突发事件 (生病/工作突发)
✗ 容易燃尽 (burnout)
```

**缓解:**
- ✓ 每天定时定量 (不要加班)
- ✓ 周末有轻松活动 (不全力冲刺)
- ✓ 如果某天无法完成，放弃该天，继续下一天
- ✓ 优先级: Week 2写作 > 其他

---

### 2. 学习质量风险 🟠 中

```
✗ 时间紧张，可能囫囵吞枣
✗ 无法深度理解，只会套用模板
✗ 容易疲劳，后期学习效率下降
✗ 没有时间做大量练习
```

**缓解:**
- ✓ Week 2专注写作 (最高ROI)
